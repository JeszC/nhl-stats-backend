/schedule

Endpoints for season schedules and game information.

Endpoints:

  • /getSchedule/{season}/{team}
    • season: start year and end year as one string (as YYYYYYYY)
    • team: three-letter team abbreviation
  • /getGame/{gameID}
    • gameID: game ID (e.g. 2023020001)
  • /getSeasonDates/{season}
    • season: start year and end year as one string (as YYYYYYYY)
