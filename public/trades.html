/trades

Endpoints for trade related data.

Endpoints:

  • /getTrades/{pageOffset}
    • pageOffset: An integer representing the 'page' that contains the trades. A bigger number represents earlier trades. The argument should be in multiples of ten, e.g., 0, 10, 20, etc. For example, number 0 would give the ten latest trades and number 10 would give the ten penultimate trades.
