/teams

Endpoints for team related data. GetTeams-endpoint only returns a list of teams, while the getTeam-endpoint returns actual team related information (rosters, season results, club history etc.).

Endpoints:

  • /getTeams/{season}
    • season: start year and end year as one string (as YYYYYYYY)
  • /getTeam/{team}/{season}
    • team: three-letter team abbreviation
    • season: start year and end year as one string (as YYYYYYYY)
