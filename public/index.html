NHL Stats Server

See each of the links below for the supported endpoints.

All endpoints will return either a JSON object, or an array of JSON objects.

In the case of the result data not existing or the query parameters being incorrect, a plain text error message is returned.

The parameters and their correct format are explained under the endpoints. Parameters are surrounded by curly braces. Optional parameters are followed by a question mark.
